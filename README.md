# TelegramBot - Newscopia

## Overview
TelegramBot-Newscopia is a functional Telegram bot developed as an individual assignment for a real-time programming course. The bot is capable of fetching and displaying news from a freely available API based on user requests.

## Features
- **News Fetching**: Retrieves news articles based on user queries.
- **Telegram Integration**: Built using Java and Maven, and deployed on AWS EC2 (currently not active due to AWS free tier limits).

## Link for the Telegram Bot
https://t.me/s286130_bot

Note: The program is deployed on an AWS VPS. (Currently not active due to AWS free tier limits)<br />
If you want to run the program on a local machine, the BOT_TOKEN and API_KEY need to be set as system environment variables.<br />

## Link for the API
https://gnews.io/
